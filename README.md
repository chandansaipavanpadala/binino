# binino
Binino is a universal toolkit designed to bridge the gap between compiled firmware and high-level source code. It facilitates binary extraction, firmware cloning, and machine code analysis across diverse microcontroller architectures and IDEs. A vital resource for hardware recovery and research.

## Phase 1: Hardware Bridge & Live Terminal

This phase establishes direct browser-to-hardware serial communication using the Web Serial API.

### Get Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
